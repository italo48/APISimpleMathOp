# APISimpleMathOp
API RESTFul simples que faz as 4 operações básicas da matemática.

Link do projeto no Heroku: [APISimpleMathOp](https://api-simple-math-op.herokuapp.com/)

# Tecnologias
- Spark Framework
- Java
- Maven

# Como usar
  ## Soma
    /sum/num1/num2
  
  ## Subtração
    /sub/num1/num2

  ## Divisão
    /div/num1/num2
  
  ## Multiplicação
    /mul/num1/num2
